Welcome to the Admin Home Page.
Below is a brief outline of all the available pages and features available for admins

Manage Users: This page contains list of users and gives admin ability to remove users

Create Quiz: This page allows admins to create a new quiz

Manage Quizzes: This page allows the admin to hide quizzes from users and remove quizzes

Warning : Removing quizzes will also remove the user results, and questions related to that quiz

View Answers: Gives admin the ability to view user's answers. Sorted by Quiz>User>User's Answers
